Wireflux

RAG Reference Pack

Pack de référence prêt à ingérer pour les réponses IA sur l'analyse réseau. Objectif: maximiser la fiabilité, citer les sources, et limiter les hallucinations.

Fichiers inclus

  • wireflux-reference-corpus.jsonl: corpus principal prêt à indexer (généré automatiquement).
  • sources.json: catalogue officiel des sources à indexer.
  • chunk-schema.json: contrat JSON pour chaque chunk.
  • chunking-policy.md: règles de découpage/document metadata.
  • prompt-policy.md: policy anti-hallucination et citation.
  • example-wireflux-corpus.jsonl: exemple d'ingestion.

Commande de régénération: npm run rag:build

Contrat de réponse recommandé

Règle Application
Evidence-first Ne répondre qu'avec des éléments présents dans les chunks récupérés.
Attribution stricte Ajouter source_id et doc_id pour chaque point clé.
Abstention Si preuve insuffisante, dire explicitement ce qui manque au lieu d'inventer.
Inférence balisée Préfixer toute déduction par "Hypothèse" + niveau de confiance.
